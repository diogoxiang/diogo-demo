
## dio-cli 前端模版 

### 项目目录

    components   (用来放内部编写的一些公有函数)
        --common
        --host-config.js    （配置调试IP）
        --url.js    （配置项目内页面路径）
        --common.js     （一些基础功能能函数封装）
        --widget    （用来存放一些插件）
    static   (用来存放外部获取的库,如jquery，bootstr)
        --js
        --styles
    page  (页面)
    build  (输出文件夹)
